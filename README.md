## Stop Watch - Timer	
This task is given in an interview. 
### Requirements: 
	1. Create Buttons ->  Start, Stop, Reset
	2. Take Timer as Input 
	3. Display Counter Value
	4. Display only Sec and in reverse order
	5. Design complete app CSS
